# 🛒 AMS Stores POS System

A lightweight, browser-based Point of Sale (POS) system designed for small retail businesses. This project focuses on simplicity and offline functionality using modern web technologies.

## 🌟 Features
- **Product Management:** Add and manage items with ease.
- **Sales Tracking:** Keep track of your daily transactions.
- **LocalStorage Integration:** All your data stays on your device—no internet required.
- **Responsive Design:** Works smoothly on both mobile and desktop browsers.
- **Receipt Generation:** Simple formatting for printing or viewing sales records.

## 🛠️ Built With
- **HTML5:** Semantic structure for the interface.
- **CSS3:** Custom styling for a clean, professional look.
- **JavaScript (Vanilla):** Logic for calculations and data handling.
- **Web Storage API:** For persistent data storage without a database.
